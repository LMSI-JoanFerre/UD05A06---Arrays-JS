# Ejercicio 9

```
Este código ordena un array de números en orden ascendente utilizando el algoritmo de selección. Selecciona repetidamente el elemento más pequeño y lo coloca en su posición correcta. Al final, muestra el array ordenado en la consola

Para ello en cada iteración del bucle externo (for), se busca el índice del elemento más pequeño a partir de la posición actual hasta el final del array. Si se encuentra un elemento más pequeño, se actualiza minIndex. Luego, si minIndex no es igual a i, se intercambian los elementos en las posiciones i y minIndex.

Este proceso se repite hasta que todos los elementos estén en su posición correcta, lo que resulta en un array ordenado.
```
